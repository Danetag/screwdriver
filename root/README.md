## Quickstart
To setup this project on your local machine:

- Gulp
```
$ npm install -g gulp
```

- Bundler (Bundle Gems)
```
$ gem update --system
$ gem install bundler
```

- NPM + Gems dependencies
```
bundle install
npm install
```
